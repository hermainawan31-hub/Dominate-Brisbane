Dominate Brisbane is a comprehensive, fully responsive gym management website designed for modern fitness centers. This professional template provides a complete digital presence for gyms, fitness studios, and health clubs with an intuitive interface and powerful features.

The website serves as both a marketing platform and a functional management tool, helping gyms attract new members, showcase their services, and streamline operations through an all-in-one management system.

✨ Key Features
🏋️ Core Pages
Home Page - Hero slider with call-to-action, service highlights, class showcase, pricing plans, and team gallery

About Us - Company overview, video introduction, skill bars, team section, and testimonials

Classes - Detailed class information with trainer profiles and timetable

Services - Comprehensive service listings with pricing plans

Our Team - Trainer profiles with social media integration

BMI Calculator - Interactive tool for health assessment

Contact - Contact form with Google Maps integration

Gallery - Responsive image gallery with lightbox

🛠️ Management Features
Smart Membership Management - Track registrations, renewals, attendance, and payments

Healthy Nutrition Plans - Customized meal planning for fitness goals

Professional Training Plans - Personalized workout schedules

Progress Tracking - Real-time performance monitoring and support

🎨 Design Highlights
Dark, modern aesthetic with orange accent colors

Fully responsive across all devices

Mobile-optimized off-canvas menu

Smooth animations and transitions

Clean typography using Google Fonts (Muli & Oswald)

Professional iconography with Font Awesome & Flaticon

📱 Technical Features

Owl Carousel for sliders

Cross-browser compatible

🚀 Installation
Clone the repository

bash
git clone https://github.com/yourusername/dominate-brisbane.git
Navigate to the project directory

bash
cd dominate-brisbane
Open in your browser

Simply open any .html file in your preferred browser

Or use a local development server (recommended)

Using VS Code with Live Server:

bash
# Install Live Server extension, then right-click any HTML file and select "Open with Live Server"
Using Python:

bash
# Python 3
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000
Using Node.js:

bash
npx serve
🎯 Customization Guide
1. Branding
Replace Gymlogo.png and last.png with your gym's logo

Update the title tag in each HTML file

Modify the brand name "Dominate Brisbane" throughout the site

2. Colors
Edit css/style.css to change the primary color:

css
/* Search for #f36100 and replace with your brand color */
.primary-btn { background: #your-color; }
.cs-item span { color: #your-color; }
/* ... and more */
